= 删除钩子源
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/hookSources/{hookSource_id}`#

删除指定的 Hook Source API 资源。



== 参数

[cols="5*"]
|===
| 名称 | Type | 在 | Required | Description 


| account_id | string | path | True  a| 
包含帐户资源的 ID

* 示例：{{.Account}}




| hookSource_id | string | path | True  a| 
要列出的 hookSource 集合的 ID

|===


== 响应

[source]
----
Status: 204, Indicates if the hookSource resource was deleted.
----
