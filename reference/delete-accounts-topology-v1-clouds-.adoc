= 删除云
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/accounts/{account_id}/topology/v1/clouds/{cloud_id}`#

删除请求的云实例以及与该特定云关联的所有非托管群集和卷。



== 参数

[cols="5*"]
|===
| 名称 | Type | 在 | Required | Description 


| account_id | string | path | True  a| 
包含帐户资源的 ID

* 示例：{{.Account}}




| cloud_id | string | path | True  a| 
要列出的云集合的 ID

|===


== 响应

[source]
----
Status: 204, Indicates if the cloud resource was deleted.
----
