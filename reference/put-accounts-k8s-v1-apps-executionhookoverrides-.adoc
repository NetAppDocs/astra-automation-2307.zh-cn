= 修改执行钩子覆盖资源
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-put]#PUT# [.api-doc-code-block]#`/accounts/{account_id}/k8s/v1/apps/{app_id}/executionHookOverrides/{executionHookOverride_id}`#

将整个存储资源替换为提供的 JSON 请求正文，同时保留不可由用户修改的键值。

*副作用*

* 如果在替换操作期间将 enabled 的值从"true"更改为"false"，则将禁用应用于指定应用程序且 hookType 值为"netapp"的任何执行挂钩。
* 如果在替换操作期间将 enabled 的值从"false"更改为"true"，则将启用适用于指定应用程序且 hookType 值为"netapp"的任何执行挂钩。



CAUTION: 此操作已弃用。



== 参数

[cols="5*"]
|===
| 名称 | Type | 在 | Required | Description 


| account_id | string | path | True  a| 
包含帐户资源的 ID

* 示例：{{.Account}}




| app_id | string | path | True  a| 
包含应用程序资源的 ID



| executionHookOverride_id | string | path | True  a| 
要列出的 executionHookOverride 集合的 ID

|===


== 请求正文

将整个存储资源替换为提供的 JSON 请求正文，同时保留不可由用户修改的键值。

*副作用*

* 如果在替换操作期间将 enabled 的值从"true"更改为"false"，则将禁用应用于指定应用程序且 hookType 值为"netapp"的任何执行挂钩。
* 如果在替换操作期间将 enabled 的值从"false"更改为"true"，则将启用适用于指定应用程序且 hookType 值为"netapp"的任何执行挂钩。


[cols="4*"]
|===
| 名称 | Type | Required | Description 


| type | string | True  a| 
资源的媒体类型。定义的值为：

* "application/astra-executionHookOverride"




| version | string | True  a| 
资源的版本。定义的值为：

* "1.0"




| id | string | False  a| 
资源的全局唯一标识符。定义的值为：

* 符合 UUIDv4 架构




| executionHookID | string | False  a| 
包含执行挂钩 ID 的 JSON 字符串。定义的值为：

* 符合 Astra 标识符架构 仅允许 hookType 值为 "netapp" 的执行挂钩资源的 executionHookID。




| 启用 | string | False  a| 
指示是否为指定应用程序启用了执行挂钩的 JSON 字符串。定义的值为：

* "true" - 执行挂钩已启用，将按指定顺序运行
* "false" - 执行挂钩已禁用，无法运行




| appID | string | False  a| 
包含重写链接的应用程序的关联 ID 的 JSON 字符串。定义的值为：

* 符合 Astra 标识符架构




| 元数据 | link:#type_astra_metadata_update["type_astra_metadata_update"] | False  a| 
与资源关联的客户端和服务指定的元数据。定义的值为：

* 符合 Astra 元数据架构如果在创建时未指定，则将创建没有标签的元数据对象。如果在更新时未指定，则元数据对象的标签、creationTimestamp 和 createdBy 将保留而不进行修改。


|===

  "version": "1.0",
  "enabled": "false"
}
----
====
