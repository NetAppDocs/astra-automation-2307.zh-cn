= 删除应用程序备份
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/accounts/{account_id}/topology/v1/appBackups/{appBackup_id}`#

删除指定的 Application Backup API 资源。正在进行的备份将被取消，然后删除。

*副作用*

* 如果备份完成，备份数据将被删除。




== 参数

[cols="5*"]
|===
| 名称 | Type | 在 | Required | Description 


| account_id | string | path | True  a| 
包含帐户资源的 ID

* 示例：{{.Account}}




| appBackup_id | string | path | True  a| 
要列出的 appBackup 集合的 ID

|===

[source]

Status: 204, Indicates if the appBackup resource was deleted.
