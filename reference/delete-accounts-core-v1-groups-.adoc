= 删除组
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/groups/{group_id}`#

删除指定的 Group API 资源。

*副作用*

* 将删除与该组关联的以下资源：
* 引用该组的角色绑定




== 参数

[cols="5*"]
|===
| 名称 | Type | 在 | Required | Description 


| account_id | string | path | True  a| 
包含帐户资源的 ID

* 示例：{{.Account}}




| group_id | string | path | True  a| 
要列出的组集合的 ID

|===


== 响应

[source]
----
Status: 204, Indicates if the group resource was deleted.
