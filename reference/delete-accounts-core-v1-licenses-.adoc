= 删除许可证
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-delete]#DELETE# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/licenses/{license_id}`#

删除指定的许可证 API 资源。无法删除自动安装的评估许可证。

*副作用*

* 重新计算授权以删除通过 Entitlement API 可见的许可证授权。如果自动安装的评估许可证在删除之前存在于此系统上，则会自动将其重新应用于系统。




== 参数

[cols="5*"]
|===
| 名称 | Type | 在 | Required | Description 


| account_id | string | path | True  a| 
包含帐户资源的 ID

* 示例：{{.Account}}




| license_id | string | path | True  a| 
要列出的许可证集合的 ID

|===
