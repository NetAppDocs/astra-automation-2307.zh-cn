= 列出集群的存储类
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/accounts/{account_id}/topology/v1/clusters/{cluster_id}/storageClasses`#

返回集合中所有匹配资源的 JSON 数组。如果在请求中指定了单个字段，则项目 JSON 数组将包含按指定顺序为每个匹配资源请求的字段。



== 参数

[cols="5*"]
|===
| 名称 | Type | 在 | Required | Description 


| account_id | string | path | True  a| 
包含帐户资源的 ID

* 示例：{{.Account}}




| cluster_id | string | path | True  a| 
包含群集资源的 ID



| 包括 | string | 查询 | False  a| 
指示列出集合时应返回哪些字段。

URL 示例："include=id"、"include=id,name"

OpenAPI 示例："id"

* 示例：id,name




| limit | string | 查询 | False  a| 
指示列出集合时要返回的最大资源数。

URL 示例："limit=2"

OpenAPI 示例："2"

|===

| type | string | True  a| 
资源的媒体类型。定义的值为：

* "application/astra-storageClass"




| version | string | True  a| 
资源的版本。定义的值为：

* "1.0"
* "1.1"




| id | string | True  a| 
资源的全局唯一标识符。定义的值为：

* 符合 UUIDv4 架构




| name | string | True  a| 
包含指示存储类名称的值的 JSON 字符串。定义的值为：

* 最小长度：1
* 最大长度：255




| 配置器 | string | True  a| 
包含指示与存储类关联的置备程序的值的 JSON 字符串。定义的值为：

* 最小长度：1
* 最大长度：255




| 可用 | string | True  a| 
JSON 字符串，指示存储类是否可用于集群或是否有资格在管理时添加到集群。定义的值为：

* "available" - 存储类可供使用
* "eligible" - 当集群处于管理下时，存储类有资格被集群使用
* "ineligible" - 存储类在管理时不符合集群使用的条件




| allowVolumeExpansion | string | True  a| 
JSON 字符串，指示存储类是否允许卷扩展。定义的值为：

* "true" - 此存储类允许卷扩展
* "false" - 存储类不允许卷扩展
* "unavailable" - 未知存储类是否允许卷扩展




| reclaimPolicy | string | True  a| 
包含指示存储类的保留策略的云特定值的 JSON 字符串。定义的值为：

* 最小长度：1
* 最大长度：255




| volumeBindingMode | string | True  a| 
JSON 字符串，包含指示存储类的绑定模式的特定于云的值。定义的值为：

* 最小长度：1
* 最大长度：255




| isDefault | string | False  a| 
JSON 字符串，指示存储类是否包含默认批注。定义的值为：

* "true" - 存储类包含默认批注




| maxSnapshotCount | 数字 | False  a| 
表示支持多少个快照的 JSON 编号。此信息附加到 NetApp 提供的存储类。



| maxBackupCount | 数字 | False  a| 
表示支持多少个备份的 JSON 编号。此信息附加到 NetApp 提供的存储类。



| 价格 | string | False  a| 
JSON 字符串，指示存储类的每单位时间的货币单位。此信息附加到 NetApp 提供的存储类。定义的值为：

* 最小长度：1
* 最大长度：63




| 货币 | string | False  a| 
表示价格所用货币的 ISO 4217 三个字母代码的 JSON 字符串。此信息附加到 NetApp 提供的存储类。定义的值为：

* 最小长度：1
* 最大长度：255




| 性能 | 数字 | False  a| 
JSON 编号，以 MBps 表示性能服务级别目标 (SLO)。此信息附加到 NetApp 提供的存储类。



| 复原力 | string | False  a| 
表示存储弹性配置的 JSON 字符串。此信息附加到 NetApp 提供的存储类。定义的值为：

* "regional" - 如果给定区域不可用，该卷将变得不可用
* "multisite" - 如果给定区域不可用，卷将继续可用




| 元数据 | link:#type_astra_metadata["type_astra_metadata"] | True  a| 
与资源关联的客户端和服务指定的元数据。定义的值为：
