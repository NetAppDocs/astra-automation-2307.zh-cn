= 检索授权
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/accounts/{account_id}/core/v1/entitlements/{entitlement_id}`#

返回 JSON 响应正文中存储的 Entitlement API 资源。



== 参数

[cols="5*"]
|===
| 名称 | Type | 在 | Required | Description 


| account_id | string | path | True  a| 
包含帐户资源的 ID

* 示例：{{.Account}}




| entitlement_id | string | path | True  a| 
要列出的授权集合的 ID

|===


== 响应

[source]
----
Status: 200, Returns the stored Entitlement API resource in the JSON response body.
----
[cols="4*"]
|===
| 名称 | Type | Required | Description 


| type | string | True  a| 
资源的媒体类型。定义的值为：

* "application/astra-entitlement"




| version | string | True  a| 
资源的版本。定义的值为：

* "1.0"




| id | string | True  a| 
资源的全局唯一标识符。定义的值为：

* 符合 UUIDv4 架构




| 分配 | string | False  a| 
包含授权分配到的对象的 ID 的 JSON 字符串。仅在分配授权时存在。架构未验证字符串内容。



| 产品 | string | False  a| 
包含从许可证或订阅中提取的产品字符串的 JSON 字符串。架构未验证字符串内容。



| productVersion | string | False  a| 
包含从许可证或订阅中提取的版本字符串的 JSON 字符串。架构未验证字符串内容。



| entitlementType | string | True  a| 
包含授权类型的 JSON 字符串。这是从许可证或订阅中提取的。架构未验证字符串内容。



| entitlementValue | string | True  a| 
包含与授权关联的容量的 JSON 字符串。架构未验证字符串内容。



| entitlementConsumption | string | False  a| 
包含与当前正在使用的授权关联的容量量的 JSON 字符串。架构未验证字符串内容。



| sourceLicense | string | False  a| 
包含授权源自的许可证对象 ID 的 JSON 字符串。定义的值为：

* 符合 Astra 标识符架构




| sourceSubscription | string | False  a| 
包含授权源自的订阅对象的 ID 的 JSON 字符串。定义的值为：

* 符合 Astra 标识符架构




| validFromTimestamp | string | False  a| 
包含指示授权何时生效的时间戳的 JSON 字符串。定义的值为：

* 符合 ISO-8601 日期时间方案




| validUntilTimestamp | string | False  a| 
包含指示授权何时不再有效的时间戳的 JSON 字符串。定义的值为：

* 符合 ISO-8601 日期时间方案




| 元数据 | link:#type_astra_metadata["type_astra_metadata"] | True  a| 
与资源关联的客户端和服务指定的元数据。定义的值为：

* 符合 Astra 元数据架构
