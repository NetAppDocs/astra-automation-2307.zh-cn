= 列出应用程序的执行挂钩覆盖资源
:allow-uri-read: 


[.api-doc-operation .api-doc-operation-get]#GET# [.api-doc-code-block]#`/accounts/{account_id}/k8s/v1/apps/{app_id}/executionHookOverrides`#

返回集合中所有匹配资源的 JSON 数组。如果在请求中指定了单个字段，则项目 JSON 数组将包含按指定顺序为每个匹配资源请求的字段。


CAUTION: 此操作已弃用。



== 参数

[cols="5*"]
|===
| 名称 | Type | 在 | Required | Description 


| account_id | string | path | True  a| 
包含帐户资源的 ID

* 示例：{{.Account}}




| app_id | string | path | True  a| 
包含应用程序资源的 ID



| 包括 | string | 查询 | False  a| 
指示列出集合时应返回哪些字段。

URL 示例："include=id"、"include=id,name"

OpenAPI 示例："id"



| limit | string | 查询 | False  a| 
指示列出集合时要返回的最大资源数。

URL 示例："limit=2"

OpenAPI 示例："2"



| 筛选器 | string | 查询 | False  a| 
指示在列出集合时应匹配哪些字段以返回资源。

支持的运算符有：

* eq - 相等
* lt/gt - 小于/大于
* lte/gte - 小于或等于/大于或等于


URL 示例："filter=field%20eq%20%27value%27"

OpenAPI 示例："field eq 'value'"

|===

* 符合 Astra 标识符架构 仅允许 hookType 值为 "netapp" 的执行挂钩资源的 executionHookID。




| 启用 | string | True  a| 
指示是否为指定应用程序启用了执行挂钩的 JSON 字符串。定义的值为：

* "true" - 执行挂钩已启用，将按指定顺序运行
* "false" - 执行挂钩已禁用，无法运行




| appID | string | True  a| 
包含重写链接的应用程序的关联 ID 的 JSON 字符串。定义的值为：

* 符合 Astra 标识符架构
